## 📌 Sobre o Projeto
Este projeto é uma **landing page profissional** desenvolvida com foco em apresentação de serviços de criação de sites.  
A proposta é oferecer uma interface limpa, moderna e objetiva para atrair clientes e apresentar soluções digitais.

## ✨ Funcionalidades
- 📢 Apresentação clara dos serviços oferecidos  
- 💻 Layout moderno e organizado  
- 📱 Design responsivo para diferentes dispositivos  
- 🎯 Botão de chamada para ação (CTA)  
- 🧱 Estrutura simples e eficiente  

## 🛠️ Tecnologias Utilizadas
- HTML5  
- CSS3  

## 🎯 Objetivo
Este projeto foi desenvolvido com o objetivo de:
- Praticar criação de landing pages  
- Melhorar habilidades em HTML e CSS  
- Trabalhar design e organização visual  
- Simular um projeto real para portfólio
